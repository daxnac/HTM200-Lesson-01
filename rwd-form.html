Responsive Web Design

The creation of responsive web design has changed the world wide web. It is the best solution for multi-device/screen web development, very light, fluid and development is easy with fewer uncomplicated code. It has solved all the head aches I have encountered in all my years as a web developer.

In the past we always had to assume or research how our audiences would view our web sites. We had to rely on custom HTML and CSS that can do responsiveness. Including other tools or WYSIWYG apps like Dreamweaver, which really arenâ€™t the best at it.

After a lot of research and taking classes, responsive web design is a lot simpler than I thought. Development has never been easier, all the resources and tools are out there on the web to help you get started. Better tools and better ways with writing code. Including frameworks such as boostrap or jquery.

With a properly designed responsive web site, all you need is one, and just that one for multiple devices and screens.
